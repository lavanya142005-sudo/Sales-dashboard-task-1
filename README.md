Sales Performance Dashboard | Power BI

Project Overview

This project presents an interactive Sales Performance Dashboard built using Power BI. The dashboard transforms raw sales data into meaningful visualizations, enabling users to monitor business performance, identify trends, and make informed decisions.

Objectives

- Analyze overall sales and profit performance.
- Track monthly sales trends.
- Compare sales across different regions and product categories.
- Provide interactive insights through filters and slicers.
- Support business decision-making with clear visual analytics.

Tools & Technologies

- Power BI Desktop
- Microsoft Excel
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated Measures and KPIs)

Dataset Information

The dataset contains transactional sales records with fields such as:

- Order ID
- Order Date
- Region
- Product Name
- Category
- Sales Amount
- Profit
- Quantity Sold

Dashboard Highlights

- 📈 Total Sales KPI
- 💰 Total Profit KPI
- 📅 Monthly Sales Trend Analysis
- 🌍 Region-wise Sales Distribution
- 🛍️ Category-wise Performance
- 🔍 Interactive Slicers for dynamic filtering

Key Insights

- Sales patterns can be analyzed month by month to identify growth trends.
- Regional comparisons help highlight high-performing markets.
- Product categories can be evaluated based on revenue and profitability.
- Interactive visuals make it easy to explore the data from multiple perspectives.

Data Preparation

Before building the dashboard, the dataset was cleaned by:

- Removing duplicate records
- Handling missing values
- Formatting date fields
- Creating calculated columns and measures
- Ensuring data consistency for reporting

How to View the Dashboard

1. Download the ".pbix" file from this repository.
2. Open it using Power BI Desktop.
3. Refresh the data if necessary.
4. Interact with charts, KPIs, and slicers to explore insights.
